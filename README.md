## Design pattern Singleton

Neste repositório tem um caso de uso de uma implementação de código utlizando o padrão Singleton. Este padrão tem como objetivo principal prover apenas uma instância de objetos em N chamadas à classe, ou seja, ele deve compartilhar um objeto para toddos que chamarem. Para permitir isso, ele faz uso de um atributo e um método estático. Este exemplo mínimo de código para representar o padrão, consiste em uma aplicação responsável por estatísticas de notas de avaliações acadêmicas.   

Stacks do projeto:

- Spring boot 4.0.4
- Maven
- Java 21
